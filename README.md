#	JobQueue API - Go Worker Pool

API REST de alta concurrencia construida en Go puro,
demuestra el poder de las goroutines y channels
para procesar trabajos en paralelo

##	¿Por qué se Eligio Go?
Go se caracteriza por manejar concurrencia masiva con pocos recursos. 
Esta API corre con 5 workers en paralelo
usando solo **2.4MB de RAM**.
En otros lenguajes de programación correria en 50-100MB
